# 🩸 Blood Donation Portal

This project is a web-based application built using **Django** that helps manage blood donor information and connects donors with those in need.

## 📝 Overview
This project allows users to register as blood donors, search for blood groups based on location, and manage donor records efficiently. It uses a clean UI and a robust Django backend.

## ✨ Steps
1. **User Registration:** Donors can sign up with their details.
2. **Blood Group Search:** Easily find donors by blood group.
3. **Admin Dashboard:** Manage and verify donor information.
4. **Validation:** Form validations for age, phone numbers, etc.
5. **Success Messages:** Real-time feedback for user actions.

## 🛠️ Tech Stack Used
- **Backend:** Django (Python)
- **Frontend:** HTML5, CSS3, Bootstrap
- **Database:** SQLite (Development) / AWS RDS (Production)
- **Hosting:** AWS EC2
- **Web Server:** Nginx & Gunicorn
